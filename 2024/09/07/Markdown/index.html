Grammar of Markdown

一 · 常见格式

  1. 大标题： 几级标题就用几个”+”+空格+文本；

  2. 小标题：

    • 无序标题：”*”+空格+文本；
    • 有序标题：数字+点+空格+文本；

  3. 单选框： 横杠+空格+ [ + 空格 + ] + 空格;

  4. 删除线：“”+文本+“”；（均为两个“~”）

  5. 表情​：​”:”​+英文单词+”:” :smile:

  6. 字体加粗：“”+文本+“”； （均为两个*）

  7. 斜体： ““+文本+”“; (单个*)

  8. 输出表格：”|” + 空格 + 文本 + 空格 + “|”

    Language C C++ python Java

  9. 插入网址：[连接名]+(网址)+空格； 百度

  10. 插入图片：！+[图片名]+(图片地址)+空格；

  11. 数学公式：”$$”+Enter；
